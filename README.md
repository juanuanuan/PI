# Programação Imperativa (Imperative Programming)
 
Este repositório contém os exercícios e trabalhos realizados no âmbito da unidade curricular de **Programação Imperativa**.

O repositório inclui as fichas práticas resolvidas ao longo do semestre, juntamente com as 100 questoes, com exceção das fichas de stacks e queues, que ainda não estão disponíveis aqui.
